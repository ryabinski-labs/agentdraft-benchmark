# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as its code of
conduct. The full, authoritative text is published at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating, you are expected to uphold this code.

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be
reported to the project maintainers at **hello@agentdraft.io**. All complaints
will be reviewed and investigated promptly and fairly. Maintainers are
obligated to respect the privacy and security of the reporter of any incident.
