# Security Policy

This repository is a standalone, read-only benchmark client for the public
AgentDraft API. It holds no secrets.

## Reporting

Report any security concern privately to **hello@agentdraft.io**. Please do not
open a public issue for a suspected vulnerability, and never paste a live API
key (`avs_live_…`) into an issue, PR, or report — rotate it from your AgentDraft
dashboard if one is exposed.
